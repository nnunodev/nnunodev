**[ravn](https://github.com/nnunodev/ravn)** — AI agent distribution framework  
`go`

Drop into any repo. Agents pick up work via Linear tickets. Async, headless, fast.

```go
// Agent picks up ticket, runs task, reports back
ravn dispatch --agent=code-review --ticket=LIN-123
```
